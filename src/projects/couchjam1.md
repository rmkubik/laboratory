[Couchjam](https://words.ryankubik.com/notes/Couchjam) is a cool idea from a game dev called Sunil. They got together with some friends and each made a local multiplayer game. They brought them together and shared. It seems like great fun.

I'm doing the same with some friends. I wanted to make something more involved in Godot, but timing wise I really need to make the simplest possible thing I can think of.

For this reason, I started checking out the Gamepad API in browser instead and got a controller connected. I'm planning to make a simple grid based game... We'll see how it comes along.
